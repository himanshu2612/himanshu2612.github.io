Access CC linux server files from your room
August 11, 2017

Using ubuntu, follow these steps

1) Open nautilus file explorer (default ubuntu file explorer) and click file->Connect to server
2) In Server address write "sftp://webhome.cc.iitk.ac.in". If it does not work, write "ssh://webhome.cc.iitk.ac.in"
3) Type cc username password when prompted.

You can access any other ssh server using this method on linux. In windows, you can use any other SSH client like WinScp or filezilla.
